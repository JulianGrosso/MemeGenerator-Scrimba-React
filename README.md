# Meme Generator - Scrimba project

_Disclaimer: This project was done for educational purposes only._

## Stack

- React
- Styled Components

## Descrition

- This project was done as part of [Scrimba](https://scrimba.com/)'s React course.
- It's desktop and mobile ready.
